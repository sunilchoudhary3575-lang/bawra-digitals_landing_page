# Assets Directory

This directory contains all images, logos, and media files for the Bawra Digitals landing page.

## Directory Structure

```
assets/
├── logos/          # Client logos (8 placeholder images)
├── gallery/        # Gallery images (6 placeholder images)
├── testimonials/   # Testimonial images (optional)
└── logo.png        # Main Bawra Digitals logo
```

## Replacing Placeholder Images

### Logo
- Replace `logo.png` with your actual Bawra Digitals logo
- Recommended size: 200x60px (or maintain aspect ratio)
- Format: PNG with transparent background preferred

### Client Logos
- Replace files in `logos/` directory
- Recommended size: 200x100px (maintain aspect ratio)
- Format: PNG with transparent background
- Files: `client-1.png` through `client-8.png`

### Gallery Images
- Replace files in `gallery/` directory
- Recommended size: 1200x1200px (square format)
- Format: JPG or PNG
- Files: `gallery-1.jpg` through `gallery-6.jpg`

### Testimonial Images (Optional)
- Replace files in `testimonials/` directory if you have client photos
- Recommended size: 400x400px (square format)
- Format: JPG or PNG

## Image Optimization

Before uploading final images:
1. Compress images using tools like TinyPNG or ImageOptim
2. Use WebP format for better performance (with fallbacks)
3. Ensure images are optimized for web (not too large file sizes)

## Current Placeholders

All current images are placeholders. The page will display placeholder text or default images if the actual files are not found.
